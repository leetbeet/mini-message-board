# Mini Message Board

A message board built with Node.js, Express, and EJS.

## Features

- Post new messages with author name
- View all messages on the homepage
- View individual messages on separate pages
- Simple, clean layout

## Technologies Used

- Node.js
- Express.js (routing, middleware)
- EJS (templating engine)
- HTML5 & CSS3 (Flexbox layout, basic styling)
